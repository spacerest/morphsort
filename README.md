# MorphSort

MorphSort is a language-learning puzzle game: sort colored rectangles to assemble morphemes into words.

**[How to play: visual guide](how-to-play/README.md)**

## Issues & feedback

This repo is for tracking issues and feedback related to the MorphSort app.

You can use it to report:
- App bugs: crashes, UI issues, unexpected behavior
- Word/morpheme issues: incorrect definitions, language issues
- Enhancement requests: new language pairs, new features
